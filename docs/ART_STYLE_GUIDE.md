# Philosophy Quest: Art Style Guide

**Visual Bible for Consistent Art Direction**

---

## OVERVIEW

**Style**: Comic Book Illustration meets Sierra Adventure  
**Vibe**: 1980s college comedy with philosophical depth  
**Reference**: Leisure Suit Larry + Day of the Tentacle + Modern indie comics

## CHARACTER DESIGN

### Brad (Protagonist)

**Reference Asset**: `char_brad.png`

#### Silhouette
- Broad shoulders, athletic build
- Slight forward hunch (thinking pose)
- Recognizable from outline

#### Outfit
- **Polo shirt**: Teal blue, two-button
- **Cargo shorts**: Khaki/olive, multiple pockets
- **Baseball cap**: Worn, slightly askew, university logo
- **Wristbands**: Athletic tape on one wrist
- **Shoes**: Sneakers (white, scuffed)

#### Facial Features
- Strong jaw, friendly face
- Slightly confused expression (default)
- Eyes that can show sudden insight
- 5 o'clock shadow (rough weekend)

#### Expressions
| Expression | Use Case |
|------------|----------|
| Confused | Default, learning |
| Panicked | Realizing deadline |
| Insight | Understanding philosophy |
| Smitten | Talking to Alex |
| Determined | Thesis defense |

#### Poses
- **Idle**: Hands in pockets, slightly slouched
- **Thinking**: Hand on chin, head tilted
- **Excited**: Arms up (touchdown pose)
| **Nervous**: Fidgeting with cup/cap

### The Four Philosophers

#### Plato (Shadow)
**Status**: To Create  
**Style**: Shadow puppet aesthetic

- **Form**: Silhouette on cave wall
- **Details**: Toga outline, beard, gesturing hand
- **Animation**: Slight flicker (torchlight effect)
- **Color**: Black silhouette on warm wall

#### John Stuart Mill (Conductor)
**Status**: To Create  
**Style**: Victorian conductor

- **Uniform**: Navy blue, brass buttons, cap
- **Props**: Lever (always), pocket watch
- **Features**: Kind face, distressed expression
- **Posture**: Slightly hunched from weight of decisions

#### Coach Williams / Nietzsche
**Status**: To Create  
**Style**: Football coach + philosophical intensity

- **Outfit**: Track suit (vintage, 80s style), whistle
- **Features**: Intense eyes, shaved head or short hair
- **Posture**: Athletic, power stance
- **The Reveal**: When unmasked, more dramatic lighting

#### Jean-Paul Sartre (Barista)

- **Shirt**: Striped, sleeves rolled
- **Features**: Judging eyes, slight smirk
- **Props**: Coffee cup, milk steamer
- **Posture**: Leaning on counter, evaluating

## BACKGROUND DESIGN

### General Principles

1. **Depth**: Foreground, midground, background layers
2. **Hotspots**: Interactive objects visually distinct
3. **Atmosphere**: Each location has unique lighting
4. **Details**: Small jokes in background (misspelled graffiti, funny books)

### Specific Locations

#### Frat House (`background_frat_party.png`)
**Completed** — Use as reference for style

- **Lighting**: Party lights (colored), TV glow
- **Details**: Red cups everywhere, pizza boxes, philosophical graffiti
- **Interactive**: Green liquid (glows slightly), phone, book
- **Easter eggs**: "Kant Touch This" graffiti (misspelled)

#### Miller's Office (`background_prof_millers_office.png`)
**Completed** — Use as reference for style

- **Lighting**: Green banker's lamp, warm, dim
- **Details**: Stacks of books, philosopher bust, rotary phone
- **Atmosphere**: Cluttered but cozy, smells like old books

## ANIMATION GUIDELINES

### Character Animation

**Frame Rate**: 12 fps (retro feel)  
**Style**: Limited animation, smear frames for action

#### Idle Animations
- **Brad**: Shifts weight, looks around, checks phone
- **Alex**: Reads book, sips coffee, adjusts glasses
- **Miller**: Adjusts glasses, sips whiskey (discreetly)
- **Dean**: Checks tablet, taps pen

#### Talk Animations
- Mouth flaps (simple shapes: O, A, M)
- Hand gestures for emphasis
- Head bobbing

#### Walk Cycle
- 8 frames
- Slight bounce
- Arms swing opposite legs

### Special Animations

#### Death Scenes
- Dramatic fall/zoom
- Freeze frame
- Color invert or fade to black
- Text: "GAME OVER" with humorous epitaph

#### Scene Transitions
- Fade to black (classic)
- Optional: Wipe transitions (retro)

#### Tecmo Bowl Epilogue
- 8-bit sprite animation
- Simple walk cycle
- Victory pose

## UI DESIGN

### Parser Interface

#### Input Box
- **Style**: Retro computer terminal
- **Color**: Black background, green text (Matrix optional, classic preferred)
- **Font**: Monospace, pixel-perfect
- **Prompt**: "> " (classic Sierra prompt)

#### Response Area
- **Style**: Text appears below input
- **Color**: White text on semi-transparent black
- **Font**: Clean sans-serif for readability
- **Scroll**: Up to 10 lines visible

#### Suggestion System (Optional)
- Ghost text showing possible completions
- Faded gray color

### Inventory Panel

#### Layout
- Grid: 2x4 or 3x3
- Thumbnails of items
- Hover/click for description

#### Style
- Background: Dark brown leather texture
- Borders: Gold/brass details
- Items: Slight shadow for depth

### Dialog Boxes

#### Character Portraits
- **Size**: 200x200px (left side)
- **Style**: Character art with expression
- **Background**: Subtle color coding per character

#### Text Area
- **Size**: Remaining width
- **Style**: Paper scroll texture
- **Font**: Serif for readability
- **Animation**: Typewriter effect (optional)

#### Choice Buttons
- Style: Paper buttons
- Hover: Slight lift shadow
- Selected: Pressed appearance

### Death Screen

#### Layout
- Center: Large "GAME OVER" text
- Below: Humorous epitaph
- Bottom: [Try Again] [Hint] buttons

#### Style
- Dark red background
- Heavy black border
- Ominous font

## FONT SPECIFICATIONS

### In-Game Text
- **Primary**: `VT323` or similar monospace (retro terminal)
- **Dialog**: `Merriweather` or similar serif (readable)
- **UI**: `Roboto` or similar sans-serif (clean)

### Titles/Headers
- **Logo**: Custom or `Press Start 2P` (retro gaming)
- **Chapter titles**: `Cinzel` (academic/classic)

### Tecmo Bowl Epilogue
- **Font**: 8-bit pixel font (NES style)
- **Reference**: Actual Tecmo Bowl typography

## TECHNICAL SPECS

### File Formats
- **Characters**: PNG, 832x1248px, transparent
- **Backgrounds**: PNG, 1280x720px
- **Items**: PNG, 1024x1024px, transparent
- **UI**: PNG or SVG, various sizes
- **Animation**: Sprite sheets or individual frames

### Color Space
- **sRGB** for consistency
- **Indexed color** optional for retro feel (16-64 colors)

### Optimization
- **Texture atlases** for sprites
- **Lazy loading** for backgrounds
- **Progressive loading** for web deployment
